Ever wanted to have your own custom spray in a game of Counter-Strike?
Have you ever tried figuring out how to make one and just been confused by the requirements?

Easy Decal creates custom sprays (or "decals") in seconds.

Even better, you don't have to follow any complex instructions to prepare your image to become a spray: Easy Decal does all the work for you.

You can create decals from any image.
All you have to do is drag-and-drop.


* The current version of Easy Decal does not install sprays into your games for you, but easy-to-follow instructions are included.
The next major upgrade of Easy Decal will be able to install your decals automatically.

** Easy Decal can read any image format that Java can read.
This includes JPG, GIF, PNG and, if you have Java 5 installed, BMP and WBMP.
The Java Image I/O library can also be extended with plugins to read other formats.
